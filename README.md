## SMART CONTRACT DEMO

### DEPLOYMENT 
- run `npm i`
- run `npx hardhat`
- run `npx hardhat compile`
- run `npx hardhat test `
- run `npx hardhat run scripts/deploy.js`
- run `npx hardhat run scripts/deploy.js --network goerli` (ALCHEMY LIVE NETWORK)
